**Total Tested:** 40+ email clients across desktop, web, and mobile platforms

### Layout & Styling Architecture

- **Layout engine:** HTML tables (industry standard for email; CSS Grid/Flexbox not supported by email clients)
- **Styling approach:** Inline styles with media query overrides for responsive behavior
- **Media queries:** Used for responsive breakpoints (600px, 480px) and dark mode (`prefers-color-scheme`)
- **Fonts:** System-safe font stacks (`Arial, Helvetica, sans-serif`) â€” no reliance on web fonts
- **Reset strategy:** Margin/padding resets, VML compatibility, and Outlook-specific MSO fixes

### Responsive Breakpoints

```css
/* Mobile-first responsive approach */
@media (max-width: 600px) {
  /* Convert multi-column to single column */
  .stack-column { width: 100% !important; max-width: 100%; }
  /* Adjust spacing for small screens */
  .mobile-padding { padding: 10px !important; }
  /* Resize images for mobile */
  .mobile-img { height: auto !important; max-width: 100%; }
  /* Hide non-essential elements */
  .mobile-hide { display: none !important; }
  /* Show mobile-only content */
  .mobile-only { display: block !important; }
}
```

### Dark Mode Support

All templates support dark mode via `prefers-color-scheme` media query:

```css
@media (prefers-color-scheme: dark) {
  .dark-mode-bg { background-color: #1a1a1a !important; }
  .dark-mode-text { color: #e0e0e0 !important; }
}
```

Compatible with clients supporting dark mode (Gmail, Apple Mail, Yahoo Mail, etc.)

## ğŸ¨ Design Patterns & Techniques

### Responsive Columns
Multi-column layouts that stack to single column on mobile using `@media` queries and `.stack-column` utility class.

### Hero Images
Full-width hero sections with responsive scaling, fallback colors, and alt text for accessibility.

### Product Grids
2/3/4-column product showcases that adapt to single column on mobile, with proper spacing and alignment.

### CTAs & Buttons
Bulletproof button technique combining VML (Outlook) and HTML/CSS anchors for maximum compatibility.

### Social Icons
Icon row implementations with proper spacing, alt text, and mobile optimization.

### Tables & Data
Data-heavy layouts (pricing, rewards, comparisons) with proper table semantics and responsive fallbacks.

### Gradient Backgrounds
CSS gradient backgrounds with solid color fallbacks for older email clients.

### Dark Mode Themes
Dual-color schemes supporting both light and dark mode preferences with CSS variables.
